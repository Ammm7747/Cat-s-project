@startuml
actor Пользователь as "Пользователь"
participant Фронтенд as "Фронтенд"
participant Бэкенд as "Бэкенд"
database БазаДанных as "База Данных"
control СервисАутентификации as "СервисАутентификации"

Пользователь -> Фронтенд: Отправляет запрос на регистрацию (логин/номер телефона)
Фронтенд -> Бэкенд: Пересылает запрос на регистрацию
Бэкенд -> БазаДанных: Проверяет, существуют ли такие данные
БазаДанных --> Бэкенд: Возвращает результат проверки

alt Регистрационные данные уникальны
Бэкенд -> СервисАутентификации: Проверяет регистрационные данные
СервисАутентификации --> Бэкенд: Возвращает результат проверки
Бэкенд -> БазаДанных: Сохраняет нового пользователя
БазаДанных --> Бэкенд: Подтверждение сохранения
Бэкенд --> Фронтенд: Отправляет успешный ответ\n(регистрация успешна)
Фронтенд --> Пользователь: Успешная регистрация

else Регистрационные данные не уникальны
Бэкенд --> Фронтенд: Отправляет ошибку\n(Пользователь уже существует)
Фронтенд --> Пользователь: Отображает сообщение об ошибке (Пользователь с такими данными уже существует)
end
@enduml
